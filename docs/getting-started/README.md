## Prerequisites

- **Python 3.10+**
- **uv** (recommended) or pip
- **Git**
- **Docker** (optional, for code execution sandbox)

## Quick Install

```bash
# Clone repository
git clone https://github.com/docxology/codomyrmex.git
cd codomyrmex

# Install with uv
uv sync

# Verify installation
codomyrmex check
```
