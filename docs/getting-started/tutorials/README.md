## Overview

Step-by-step tutorials for learning Codomyrmex. From creating your first module to advanced AI integration patterns.

| Tutorial | Description | Time |
|----------|-------------|------|
| [**Creating a Module** (overview)](creating-a-module.md) | End-to-end guide: scaffold, implement, test, document | 2 hr |

### 2. PAI Integration
