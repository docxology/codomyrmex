# Personal AI Infrastructure — Bio-Inspired Design

**Version**: v1.2.3 | **Status**: Active | **Last Updated**: March 2026

## Overview

Bio-inspired design documentation supports PAI by providing the conceptual framework that informs Codomyrmex's approach to personal AI infrastructure. The PAI-Codomyrmex relationship is itself a **symbiosis** — PAI provides cognitive architecture (goal decomposition, agent delegation, adaptive planning), while Codomyrmex provides execution infrastructure (modules, tools, APIs). Together they form a **holobiont** whose capabilities exceed either partner's in isolation. See [symbiosis.md](./symbiosis.md) for the full theoretical treatment.

## Trust Gateway as Social Immunity

The PAI Trust Gateway (UNTRUSTED → VERIFIED → TRUSTED) mirrors the 3-tier immune system described in [immune_system.md](./immune_system.md):

- **UNTRUSTED** = Epithelial barrier — nothing enters
- **VERIFIED** = Innate immunity — pattern-matched safe tools promoted
- **TRUSTED** = Adaptive immunity — specific destructive tools enabled after explicit validation
