# PAI Screenshots Specification

**Version**: v1.1.9 | **Status**: Active | **Last Updated**: March 2026

## Overview

Static screenshot assets documenting the PAI (Personal AI Infrastructure) system interface, integration points, and visual workflows.

## Functional Requirements

1. Provide visual reference for PAI interface components (analytics, board, calendar, dispatch, email, git, network)
2. Support PNG and WebP image formats for cross-platform compatibility
3. Serve as documentation assets linkable from other markdown files

## Contents

- 15 PNG screenshots covering all PAI interface tabs
- 1 WebP animated interface tour
