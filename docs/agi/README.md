# AGI Theory and Evidence-Bounded Perspectives

**Version**: v1.3.0 | **Status**: Active | **Last Updated**: March 2026

## Overview

Theoretical foundations for discussing Codomyrmex as an AGI scaffold. The essays use
formal ideas as analytical lenses, then separate code correspondence from empirical
evidence. They cover emergence, agency, tool ontology, world models, bounded change
evaluation, and the Colony Thesis; none treats the repository as an AGI system.
