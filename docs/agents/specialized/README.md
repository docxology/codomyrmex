# Specialized Code Improvement

**Module**: `codomyrmex.agents.specialized` | **Category**: Specialized | **Last Updated**: March 2026

## Overview

Autonomous code improvement pipeline with iterative review loops. Anti-pattern detection, confidence-gated fix proposals, regression test generation, and convergence-based quality improvement.

## Purpose

Autonomous code improvement pipeline and iterative review loop for detecting anti-patterns, proposing fixes with confidence-gated approval, generating regression tests, and converging on code quality through repeated generate-test-review cycles.
