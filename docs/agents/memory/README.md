# Agent Memory


**Version**: v0.1.0 | **Status**: Active | **Last Updated**: May 2026

**Module**: `codomyrmex.agents.memory` | **Category**: Core Infrastructure | **Last Updated**: April 2026

## Overview

Agent memory utilities for conversation history, lightweight key-value recall, and learning journal entries. The module backs agent workflows that need local continuity without depending on external services.

## Key Components

| Component | Purpose |
|:---|:---|
| `ConversationHistory` | Stores and exports role-based conversation turns |
| `MemoryStore` | Provides tagged, expiring in-memory records |
| `LearningJournal` | Records observations and extracts recurring patterns |
