# defense

**Version**: v1.3.0 | **Status**: Active | **Last Updated**: July 2026

## Overview

Documentation and publishing references for the Codomyrmex defense module. The
module supplies local active-defense primitives for prompt-exploit detection,
honeytokens, request screening, rate limiting, threat reports, and rabbit-hole
containment.
