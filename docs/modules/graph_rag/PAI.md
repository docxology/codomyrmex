# Personal AI Infrastructure — Graph RAG Module

**Version**: v0.1.7 | **Status**: Active | **Last Updated**: February 2026

## Overview

The Graph RAG module supports Personal AI Infrastructure through personal knowledge graph construction and retrieval-augmented generation.

## Local-First AI

Knowledge graphs stored locally; RAG queries can use local embeddings and LLMs

## PAI Capabilities

- Local knowledge graph storage
- Private document indexing
- Offline RAG queries with local models
