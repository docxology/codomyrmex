# embodiment

**Version**: v1.3.0 | **Status**: Active | **Last Updated**: July 2026

## Overview

Documentation and publishing references for the Codomyrmex embodiment module.
The module provides deterministic local interfaces for telemetry, simulated
sensors and actuators, WebSocket bridging, ROS-style pub/sub, and 3D transforms.
