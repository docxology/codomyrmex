# embodiment - MCP Tool Specification

**Version**: v1.3.0 | **Status**: Active | **Last Updated**: July 2026

## Current Status

No MCP tools are currently exported by the embodiment module. The module is a
programmatic simulation and bridge surface for local code, tests, and demos.

## Future Tool Criteria

If MCP tools are added, they must remain deterministic, local, and bounded.
Candidate future tools include telemetry summarization, transform validation, or
simulated actuator dry-run checks.
