# Multimodal Examples -- Agent Coordination

**Version**: v1.0.8 | **Status**: Active | **Last Updated**: March 2026

## Overview

Example scripts demonstrating image generation with Google AI (Imagen 3) using the codomyrmex multimodal module.

## Scripts Available

- **basic_usage.py**: Minimal pattern for text-to-image generation via Google AI Imagen 3 API


## Agent Instructions

1. Run scripts from the repository root directory using `uv run python scripts/multimodal/examples/<script>`
2. Ensure prerequisites are installed: `uv sync (requires GEMINI_API_KEY environment variable)`
3. Scripts are demonstration/utility tools and do not modify production state
