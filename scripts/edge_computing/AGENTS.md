# Edge Computing -- Agent Coordination

**Version**: v1.1.9 | **Status**: Active | **Last Updated**: March 2026

## Overview

Demonstrates the full lifecycle of edge computing including node management, function deployment, state synchronization, scheduling, and monitoring.

## Scripts Available

- **orchestrator.py**: Runs edge cluster lifecycle: node registration, function deployment, scheduling, and metrics collection


## Agent Instructions

1. Run scripts from the repository root directory using `uv run python scripts/edge_computing/<script>`
2. Ensure prerequisites are installed: `uv sync`
3. Scripts are demonstration/utility tools and do not modify production state
