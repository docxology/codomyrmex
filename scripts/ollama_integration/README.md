# Ollama Integration Scripts

Scripts for managing and using Ollama LLM integration with Codomyrmex.

## Scripts

- `basic_usage.py` - Simple Ollama model execution examples
- `integration_demo.py` - Comprehensive Ollama integration demonstration
- `model_management.py` - Ollama model management and configuration
- `orchestrate.py` - Main orchestrator for Ollama workflows

## Usage

```bash
# Basic usage
python scripts/ollama_integration/basic_usage.py

# Comprehensive demo
python scripts/ollama_integration/integration_demo.py

# Model management
python scripts/ollama_integration/model_management.py

# Main orchestrator
python scripts/ollama_integration/orchestrate.py
```

## Prerequisites

- Ollama installed and running
- Ollama models downloaded (e.g., `ollama pull llama3.1:8b`)

## Features

- Model listing and management
- Flexible execution with custom options
- Output saving and configuration management
- Model comparison and benchmarking
- Batch execution with concurrency control
- Integration with Codomyrmex ecosystem
