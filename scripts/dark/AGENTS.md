# Dark Mode Processing -- Agent Coordination

**Version**: v1.1.0 | **Status**: Active | **Last Updated**: March 2026

## Overview

Orchestrator script for the dark module demonstrating PDF dark mode processing with presets and custom filter configurations using PyMuPDF (fitz).

## Scripts Available

- **dark_orchestrator.py**: Processes PDFs with dark mode filters, supporting multiple presets and custom color configurations


## Agent Instructions

1. Run scripts from the repository root directory using `uv run python scripts/dark/<script>`
2. Ensure prerequisites are installed: `uv sync --extra dark (requires PyMuPDF/fitz)`
3. Scripts are demonstration/utility tools and do not modify production state
