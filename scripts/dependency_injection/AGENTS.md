# Dependency Injection -- Agent Coordination

**Version**: v1.1.0 | **Status**: Active | **Last Updated**: March 2026

## Overview

Demonstrates the dependency injection module including IoC container usage, scope management, and auto-injection with Protocol-based interfaces.

## Scripts Available

- **orchestrator.py**: Shows IoC container registration, scoped injection, and service resolution patterns


## Agent Instructions

1. Run scripts from the repository root directory using `uv run python scripts/dependency_injection/<script>`
2. Ensure prerequisites are installed: `uv sync`
3. Scripts are demonstration/utility tools and do not modify production state
