# .pipelines

**Version**: v1.0.8 | **Last Updated**: March 2026

## Overview

This directory contains CI/CD pipeline definitions and automation workflows for the Codomyrmex project. Pipelines are managed by the `ci_cd_automation` module and can be triggered via GitHub Actions or local execution.

## Structure

Pipeline definitions may include:

- Build pipeline configurations
- Test pipeline templates
- Deployment workflow specifications
- Release automation scripts
