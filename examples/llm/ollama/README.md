## Overview

Examples for local LLM integration using Ollama.

## Configuration

`rnj_1_8b_config.json` provides model configuration for the RNJ 1.8B model.

## Running

```bash
# Ensure Ollama is running
ollama serve

# Run examples
cd examples/llm/ollama
python example_basic.py  # if available
```
