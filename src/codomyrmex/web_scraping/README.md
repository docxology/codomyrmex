# Web Scraping Module

The Web Scraping module provides simple, standard library-only tools for web scraping.

## Available MCP Tools

- `scraping_fetch_page`: Fetches an HTML page and returns status and raw HTML.
- `scraping_extract_links`: Extracts all fully-qualified URLs from a given page.
- `scraping_get_text`: Cleans and returns raw text content from a URL without external libraries.
