# Collaboration -- Agent Integration Guide

**Version**: v1.1.9 | **Status**: Active | **Last Updated**: March 2026

## Agent Capabilities

Enables multi-agent collaboration with swarm task submission, pool monitoring, and agent listing.

## Available MCP Tools

| Tool | Description |
|------|-------------|
| `swarm_submit_task` | Submit a task to the agent swarm |
| `pool_status` | Get agent pool status |
| `list_agents` | List collaboration agents |

## Trust Level

All three MCP tools are classified as **Safe**.
