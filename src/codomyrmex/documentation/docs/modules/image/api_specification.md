# Image - API Specification

**Version**: v0.1.0 | **Status**: Reserved | **Last Updated**: March 2026

## 1. Overview
The `image` module is a reserved namespace for image processing, analysis, and format operations. Image generation lives in `codomyrmex.multimodal`; this module focuses on metadata extraction and format information.

## 2. Current State
This module is currently a reserved namespace with no public exports. Planned functionality includes image metadata extraction, format detection, and basic transformations.

## 3. Related Modules

| Module | Relationship |
|--------|-------------|
| `multimodal` | Image generation via `ImageGenerator` |
| `vision` | Visual analysis via VLM (Ollama llava/bakllava) |
