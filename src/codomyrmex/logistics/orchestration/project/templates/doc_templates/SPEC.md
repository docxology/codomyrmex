# doc_templates - Documentation Specification

**Version**: v0.1.0 | **Status**: Active | **Last Updated**: January 2026

## Purpose

Specific templates for generating documentation files (README, SPEC, AGENTS).

## Functional Requirements

1. **Compliance**: Templates must produce docs that adhere to the project standards.
2. **completeness**: Should cover all standard sections (Signposting, Overview, Navigation).
