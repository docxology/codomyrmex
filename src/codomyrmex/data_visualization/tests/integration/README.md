## Overview

Integration tests for the Data Visualization module. These tests verify the end-to-end generation of complex diagrams from live project data and metadata files.

## Test Areas

- **Full Project Visualization**: Generating comprehensive module relationship diagrams for the entire Codomyrmex repository.
- **Commit Timeline Synthesis**: Testing the extraction of git history and its transformation into visual timelines.
- **Web Export**: Verifying that generated diagrams are correctly formatted for embedding in markdown AND documentation websites.
