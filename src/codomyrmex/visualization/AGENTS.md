# Visualization Agents

**Version**: v0.1.0 | **Status**: Active | **Last Updated**: February 2026

## Overview

While primarily a utility module, Visualization provides tools for "Analyst" type agents.

## Agents

### `AnalystAgent` (Conceptual)

- **Role**: Monitors system state and generates reports.
- **Tools**: `generate_report`
- **Behavior**: Periodically checks module states (balances, colony health) and updates the dashboard.

## Future Expansions

- **AnomalyDetectionAgent**: Uses visualization data to flag outliers.
