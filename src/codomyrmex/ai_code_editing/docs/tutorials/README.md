## Overview

Practical, hands-on tutorials for mastering the AI Code Editing module. These guides walk through common developer workflows, demonstrating how to leverage droids for code generation and refactoring while maintaining project standards.

## Tutorial Topics

- **Prompt Engineering Basics**: How to craft effective task and context prompts.
- **Provider Switching**: Using different LLM backends (OpenAI, Anthropic, Gemini) seamlessly.
- **Refactoring Legacy Code**: Step-by-step guide to modernizing old functions.
- **Batch Processing**: Automating multiple code edits in a single workflow.
