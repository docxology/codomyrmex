## Overview

A collection of standardized markdown templates used for prompt engineering within the AI Code Editing module. These templates ensure that all AI interactions follow a consistent structure and include necessary security and quality constraints.

## Template Types

- **System Templates**: Define core AI behavior, role (e.g., Clean Code Expert), and safety boundaries.
- **Task Templates**: Standardized formats for describing coding objectives, parameters, and success criteria.
- **Context Templates**: Frameworks for providing relevant code snippets and project-specific environment details.
