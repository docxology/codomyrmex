# governance

## Overview

Governance capabilities for the Codomyrmex platform including contract lifecycle management, policy enforcement, dispute resolution, and compliance visualization. Provides structured workflows for managing agreements between agents, enforcing rule-based policies, and resolving disputes through a formal mediation process.
