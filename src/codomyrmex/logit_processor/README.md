# Logit Processor Module

## Purpose
The `logit_processor` module provides composable processors for language model token sampling in the Codomyrmex ecosystem.

## Features
- Provides tools for adjusting and controlling logit values during language model generation.
- Exposed via MCP tools (`process_logits`) for integration with agents and external tools.
