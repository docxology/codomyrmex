# Language Detection Module

This module provides language detection capabilities for the Codomyrmex platform.

## Overview

The language detection module exposes tools to analyze text and determine its language using the `langdetect` library.

## Available MCP Tools

- `language_detection_detect`: Detects the language of a given text and returns its ISO 639-1 code.
- `language_detection_detect_probs`: Detects possible languages for a text, returning each with a confidence probability.

## Setup

Ensure that `langdetect` is installed in your environment.
